Please follow the steps to run task runner

  • Open Developers mode and open console in Browser
  • Tere are 3 tasks created
  • To start task runner with a task use the following syntax
  • t1 = new TaskRunner(ts.task1, 50000)
  • First param: Task | ts.task1, ts.task2, ts.task3
  • Second param: Ms | Time in microsecond
  • Start task: t1.start([Parametes set comma seperated]) | t1.start('ag1', 'ag2', 'ag3')
  • Reset task: t1.reset(10000) | time in microsecond
  • Stop task: t1.stop()
